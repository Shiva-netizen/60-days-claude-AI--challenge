# Day 12 - AI-Powered Job Search & Personal Branding Toolkit

## 🎯 Objective

Learn how Claude AI can act as a Technical Recruiter, Hiring Manager, Career Coach, Executive Resume Writer, and Personal Branding Consultant to generate a complete job search toolkit from an existing resume and professional profile.

---

# 🚀 Project Overview

Modern hiring extends beyond resumes.

Recruiters evaluate:

* Resume Quality
* LinkedIn Profile
* Communication Skills
* Personal Branding
* Networking Approach
* Professional Positioning
* Interview Readiness

This project demonstrates how Claude can analyze a professional profile and generate a complete career toolkit designed to improve visibility, networking effectiveness, and interview opportunities.
