# Day 16 - Building a Custom Claude Skill for Stock Fundamental Research

## 🎯 Objective

Learn how Claude Custom Skills can transform long, repetitive prompts into reusable workflows that can be applied across multiple conversations without rewriting instructions every time.

---

# 🚀 Project Overview

One of the biggest productivity challenges when working with AI is repeatedly providing the same instructions.

For example:

Every stock analysis may require:

* Financial Statement Analysis
* Valuation Review
* Profitability Assessment
* Balance Sheet Analysis
* Ownership Review
* Peer Comparison
* Risk Evaluation

Instead of repeatedly writing detailed prompts, Claude Skills allow these instructions to be saved and reused.

This project demonstrates how to build a reusable skill called:

### stock-fundamental-research

A structured workflow for analyzing listed companies using business fundamentals.

---

# 🧠 What are Claude Skills?

Claude Skills are reusable instruction sets that allow Claude to consistently follow predefined workflows.

Benefits:

✅ Consistency

✅ Faster Analysis

✅ Reduced Prompting

✅ Better Productivity

✅ Standardized Outputs
