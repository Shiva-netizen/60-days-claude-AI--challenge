## 🎯 Objective

Learn how Claude AI can generate interactive educational games that teach prompt engineering through hands-on challenges. Instead of memorizing prompt techniques, players build, refine, and optimize prompts while receiving instant feedback on their decisions.

---

# 🚀 Project Overview

Writing effective prompts is becoming one of the most valuable skills in the AI era.

However, prompt engineering is best learned through experimentation rather than reading long tutorials.

This project demonstrates how Claude can generate a complete gamified learning experience where users solve prompt puzzles, compare AI outputs, remove unnecessary instructions, and understand why some prompts consistently produce better results.

---

# 🤖 What is Prompt Engineering?

Prompt Engineering is the practice of designing clear, structured instructions that help AI produce accurate, relevant, and high-quality responses.

A well-crafted prompt provides:

* A clear role
* Sufficient context
* Specific objectives
* Helpful constraints
* Desired output format

The better the prompt, the better the AI response.

# 🧩 Challenge 1 – Build the Prompt

Players receive a desired outcome and a collection of draggable prompt blocks.

Examples of blocks:

* Assign a role
* Add context
* Define the objective
* Specify constraints
* Choose an output format

Some blocks are intentionally misleading.

The goal is to assemble the most effective prompt.

---

# 🧹 Challenge 2 – Clean the Prompt

Players receive an overly long or poorly structured prompt.

Their task is to:

* Remove redundant instructions
* Eliminate vague wording
* Keep only what improves AI performance

The simulator demonstrates that clarity often outperforms complexity.
