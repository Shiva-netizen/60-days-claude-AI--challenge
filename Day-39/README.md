🎯 Objective

Learn how Claude AI can generate commercial-quality productivity software by building a browser-based PDF Splitter & Merger with professional UI, drag-and-drop interactions, live previews, and secure client-side document processing.

🚀 Project Overview

Managing PDF files is a common task for students, professionals, businesses, and developers.

Whether you need to extract specific pages, split large documents, or combine multiple PDFs into one, traditional online tools often require uploading sensitive documents to external servers.

This project demonstrates how Claude can generate a fully interactive PDF management application that performs all processing locally inside the browser—ensuring speed, privacy, and security.

📄 What is PDF Splitter & Merger?

PDF Splitter & Merger is a modern browser-based productivity application that enables users to split, extract, preview, rearrange, and merge PDF documents without relying on any backend services.

All operations happen client-side using JavaScript, giving users complete control over their files while maintaining privacy.
